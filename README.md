# FLASK REST API

## Description

A Python flask project of a REST API.

## Installation

1. CLone this repository
2. Open terminal and install project requirements with ```pip install -r requirements.txt```
3. Run command `python3 app.py`

## Technologies Used

- flask
- flask-restful
- flask-httpauth
- peewee
- Sqlite
- reqparse
- argon2-cffi
- itsdangerous
